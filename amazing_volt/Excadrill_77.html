Excadrill (Japanese: ドリュウズ Doryuzu) is a Metal-type Stage 1 Pokémon card. It is part of the Vivid Voltage expansion.

Trivia

Origin

Drill Run is a move in the Pokémon games that Excadrill can learn. Slashing Claw is an attack that first appeared on Type: Null from Crimson Invasion. This card's Pokédex entry comes from Pokémon Sword.
